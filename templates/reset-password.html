Reset Password

Follow these steps to reset your password

Password Reset Instructions

1 Stop the Huntarr container:
docker stop huntarr
2 Delete the credentials file from your configuration volume:
rm /path/to/your/config/user/credentials.json

For typical Docker setups, this would be something like:

rm /mnt/user/appdata/huntarr/user/credentials.json
3 Restart the Huntarr container:
docker start huntarr
4 Create a new account:

After restarting, you'll be prompted to create a new account when you access the web interface again.
