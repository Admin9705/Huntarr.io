Cleanuparr

Cleanuparr by Flaminel

View on GitHub

Cleanuparr is a powerful CLI tool designed to automatically clean up and organize your media libraries for Sonarr, Radarr, and other *arr applications.

Key Features:

  • Automatically removes duplicate files based on quality preferences
  • Cleans up orphaned files that aren't tracked by your media server
  • Identifies and manages sample videos
  • Helps manage subtitle files efficiently
  • User-friendly command-line interface with clear options

About the Author

Cleanuparr is developed by Flaminel, a passionate developer focused on creating tools that enhance the media server experience.

Huntarr is proud to feature Cleanuparr as part of our commitment to helping other projects grow. We believe in collaboration across the media server community to create better tools for everyone.
